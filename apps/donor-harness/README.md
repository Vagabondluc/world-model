# Donor Harness (minimal)

Quick scaffold for local development of the donor harness mounting donor UIs.

Install and run:

```bash
cd world-model/apps/donor-harness
npm install
npm run dev
```

This is a minimal scaffold: the canonical store is a small Zustand store in `src/store/canonicalStore.ts` and a sample donor shim is at `src/shims/adventureShim.ts`.
