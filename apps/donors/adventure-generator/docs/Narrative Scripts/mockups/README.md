# Narrative Scripts UI Mockups

This directory contains the complete UI mockup suite for the Narrative Scripts repository, covering over 70+ scripts. Each script is addressed by a set of four deliverables: Questionnaire, Wireframe, UI Documentation, and Sample Output.

## 📂 Subdirectories
| Folder | Purpose |
| :--- | :--- |
| **`questionnaires/`** | Defines the data inputs, variables, and logic requirements. |
| **`wireframes/`** | ASCII representations of the web-based interfaces. |
| **`ui-docs/`** | Technical explanations of the component-driven architecture. |
| **`samples/`** | Demonstration of the system output based on the script data. |

## 🕹️ Major Deliverables
- **Tactical Combat Studio:** A unified environment for managing arenas, balancing, and flow.
- **Character Backstory Studio:** Deep persona and narrative arc generation.
- **Living World Hub:** Faction clocks, conflict sims, and downtime management.
- **Narrative Plot Engine:** A library of 34 plot engines and 18 universal twists.
- **Investigation Suite:** Multi-node mystery campaigns and node-cloud visualizers.
