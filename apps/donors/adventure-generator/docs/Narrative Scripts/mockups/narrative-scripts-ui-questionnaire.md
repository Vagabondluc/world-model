**Owner:** TBD

**Purpose:** Design specifications for each script's UI mockup (wireframe + narrative mockup)

---

## Instructions

For each script below, answer the following questions to determine the UI design:

1. **Primary Function**: What does this script generate/output?
2. **Input Fields**: What user inputs are needed? (text, dropdowns, sliders, number fields, etc.)
3. **RNG Method**: Seed-based (80% default) OR table-based picker? Specify RNG tables if needed.
4. **AI Context**: Does AI need special context/instructions? What placeholder text?
5. **Output Format**: Markdown, JSON, plain text, HTML, stat block, narrative prose?
6. **Save/Load**: Can users save templates? Export results (JSON/CSV/TXT)?
7. **Preview**: Live preview panel or separate view?
8. **Special UI**: Any unique components (dice roller, map viewer, relationship graph, timeline)?
