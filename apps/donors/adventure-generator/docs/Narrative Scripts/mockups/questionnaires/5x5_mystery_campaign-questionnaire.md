# UI Questionnaire: 5x5_mystery_campaign

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A meta-campaign architect that coordinates five interconnected 5-node mysteries into a massive 25-node "5x5" investigative campaign with overarching narrative threads.

## 2. Core Inputs
- **Campaign Name:** The title of the overarching story.
- **Overarching Narrative:** The grand conspiracy or secret that ties all mysteries together.
- **Five Individual Mystery Concepts:** Unique themes for each sub-mystery (A, B, C, D, E).
- **Recurring Elements:** Shared NPCs, locations, or motifs across the campaign.
- **Entry Points:** Flexible start points (A, B, C, or D).

## 3. UI Requirements
- **Campaign Meta-Map:** A high-level view showing the five mysteries as "Super-Nodes" (A -> B/C/D -> E).
- **Inter-Mystery Clue Tracker:** Displays how clues in one mystery (sub-nodes) point to another mystery entirely.
- **Recurring Element Manager:** A central database for NPCs and locations shared across the 25 nodes.
- **Complexity Escalator:** A slider or setting to manage the increasing stakes and DC modifiers as the campaign progresses.
- **Flexible Entry Selector:** Tools to re-orient the campaign flow based on where players start.

## 4. Derived & Automated Fields
- **Global Clue Density Map:** Visualizes the "web" of 25 nodes and 15+ inter-mystery connections.
- **Incremental Revelation Suggester:** Suggests which campaign-level secrets to reveal in which sub-mystery.
- **Consistency Auditor (Global):** Checks for contradictions across all 5 mysteries (e.g., shared NPCs having consistent timelines).

## 5. Exports & Integration
- 5x 5-Node Mystery Packages (25 files total).
- 1x Inter-Mystery Connection Key.
- 1x Campaign Narrative Summary.
- 1x Recurring Elements Dossier.
- 1x Master Campaign Overview.
