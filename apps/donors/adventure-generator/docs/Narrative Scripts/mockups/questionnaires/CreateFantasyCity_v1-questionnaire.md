# UI Questionnaire: CreateFantasyCity_v1

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A tool to generate a detailed fantasy city (5,000–10,000 residents) with specific architectural, cultural, and historical depth, tailored to a forest/glade environment.

## 2. Core Inputs
- **Dominant Race:** The primary population group.
- **Region/Biome:** The geographical setting (e.g., Dark Timberlands).
- **Core Architectural Features:** Circles surrounding a glade, treetop housing, rope/wooden bridges.
- **Government Direction:** Preferences for leadership types.
- **Cultural Elements:** Statues, tapestries, gardens.
- **Historical Focal Points:** Creation myth, battles, coups, Luminarch history.
- **Shop Requirements:** Explicit list including Magic Shop, Armorer, Weaponsmith, Herbalist, Alchemist + 5 more.

## 3. UI Requirements
- **Settlement Profile Editor:** Fields for Name, Population, and Race Distribution.
- **Architecture Modeller:** A section to define housing styles (Treetop, Grounded, Natural Resources).
- **History & Lore Timeline:** A tool to draft major events (Battles, Coups, Leadership changes).
- **Cultural Asset Gallery:** A list to define prominent statues, gardens, and art pieces.
- **Marketplace Builder:** A grid to name and describe the 10 mandatory and suggested shops.

## 4. Derived & Automated Fields
- **NPC Demographics Generator:** Calculates population numbers per race based on the "Dominant Race" input.
- **Architectural Resource Matcher:** Automatically suggests materials based on the chosen "Timberlands" biome.
- **Luminarch Succession Logic:** Suggests historical leadership transitions.

## 5. Exports & Integration
- City Gazetteer (Markdown).
- Interactive Map Labels (VTT).
- Merchant Directory (Handout).
