# UI Questionnaire: Alternate NPC Template

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A comprehensive NPC builder focused on identity, personal history, and representative analogies (famous character comparisons) for D&D-style fantasy worlds.

## 2. Core Inputs
- **Basic Identity:** (Name, Race, Gender, Sexual Preference).
- **Physical Profile:** (Detailed fields for Skin Color, Height, Weight).
- **Professional History:** (Job, "The Why" behind the job).
- **Famous Character Analogy:** (Comparison to a book/movie character).

## 3. UI Requirements
- **Identity & Orientation Studio:** A sensitive but thorough set of fields for gender and sexual preference.
- **Physicality Matrix:** Numerical and qualitative inputs for height, weight, and skin tones.
- **Career Path Editor:** A narrative field to explain the "Origin Story" of the NPC's profession.
- **Abilities & Inventory Tracker:** Lists for skills and what the NPC "carries on themselves."
- **Analogy Engine:** A search or text field to link the NPC to a "Famous Character" for GM reference (e.g., "Like Han Solo but with magic").
- **Secret & Hook Console:** Dedicated areas for "The Hidden Secret" and "D&D Fantasy Hook."

## 4. Derived & Automated Fields
- **Voice Simulator:** Automatically generates a dialogue snippet based on personality and job.
- **Inventory Weight Calculator:** (Optional) Estimates the weight of items carried.
- **Analogy-Based Behavior Suggestions:** Provides roleplay tips based on the selected "Famous Character" comparison.

## 5. Exports & Integration
