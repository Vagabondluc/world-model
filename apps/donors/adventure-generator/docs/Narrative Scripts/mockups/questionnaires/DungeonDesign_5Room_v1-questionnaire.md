# UI Questionnaire: DungeonDesign_5Room_v1

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A streamlined 5-room dungeon generator with specific focus on the classic roles of each room.

## 2. Core Inputs
- **Seed:** For RNG generation.
- **Theme/Environment:** (e.g., "Sewer", "Space Station").
- **Room 1: Entrance:** Guard/Trap details.
- **Room 2: Puzzle/RP:** Negotiation/Problem-solving.
- **Room 3: Setback/Trick:** The twist or complication.
- **Room 4: Climax:** The big fight.
- **Room 5: Reward:** Loot and plot twists.

## 3. UI Requirements
- **Horizontal 5-Panel Scroll:** Each room as a distinct pillar.
- **Role presets:** Dropdown for "Guard Type", "Puzzle Genre", etc.
- **AI Sync:** Fill specific rooms or the whole set.

## 4. Derived & Automated Fields
- **Mood Suggestions:** Color-code each room pillar based on its narrative mood (Mystery, Intellect, Suspense, Intensity, Revelation).
