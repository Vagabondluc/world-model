# UI Questionnaire: Encounter_genric_beta

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A creative writing portal that generates immersive, 500-word "Mystical Realm" encounters with variable XP based on outcome performance.

## 2. Core Inputs
- **Realm Type:** (e.g., Feywild, Shadowfell, Astral Plane).
- **Environmental Intrigue:** The mechanical "twist" of the area.
- **Narrative Anchor:** Selection (Skill Challenge or RP Scenario).
- **Challenge Content:** Detailed text or prompts for the encounter.
- **Combat Components:** Opponent source/statblock.
- **Tactical Insight:** NPC motives and methods.

## 3. UI Requirements
- **Word-Count Focused Editor:** A text area that visually tracks the "500-word" limit.
- **XP Variable Toggle:** A switch that shows XP for "Total Success", "Partial Success", and "Failure".
- **Motives & Tactics Card:** A small sidebar for quickly referencing NPC personality during play.
- **Immersive Backgrounds:** Theme-based UI skins (Fey, Goth, Cosmic).

## 4. Derived & Automated Fields
- **Success/Failure Scenario Generator:** Prompts for different outcome results.
- **Source Linker:** Automatically looks up monster names in SRD/Open5e.

## 5. Exports & Integration
- Social Media "Dungeon Share" image (Visual snippet).
- Markdown Narrative.
