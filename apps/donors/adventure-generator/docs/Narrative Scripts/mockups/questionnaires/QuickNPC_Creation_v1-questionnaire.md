# UI Questionnaire: Quick NPC Reference Generator

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A rapid-response tool for GMs to generate succinct, roleplay-ready NPC snapshots during active sessions, focusing on immediate table-utility over complex backstory.

## 2. Core Inputs
- **NPC Identifier:** (Name or Role).
- **Core Role:** (Merchant, Guard, Informant, etc.).
- **"Unexpected Detail" Hook:** (A field to seed the script's requirement for a twist).
- **Generation Speed:** (Instant / Single-Click).

## 3. UI Requirements
- **Sucinct Fact-Grid:** A 9-category table editor (Identity, Look, Goal, Traits, Habits, Speech, Knowledge, Bonds, Cues).
- **One-Sentence Constraint Enforcer:** UI markers or validation to keep each entry brief.
- **Deepening Narrative Studio:** A separate text area for the 1-2 paragraph character deep-dive.
- **"Unexpected" Trigger:** A button/logic to randomize or manually input the character's unique twist.
- **Quick-Swap Reference View:** A "Card" mode for easy reading during a live game.

## 4. Derived & Automated Fields
- **Roleplay Cue Synthesis:** Automatically highlights the "Cues" and "Speech" sections for prominent visibility.
- **Knowledge Linker:** Suggests 1 "Secret" the NPC might know based on their role/location.
- **Consistency Checker:** Ensures the "Habits" and "Speech" sections don't contradict the "Traits."

## 5. Exports & Integration
- Markdown Quick-Ref Table.
- Digital NPC Character Card.
- Session Log entry.
