# UI Questionnaire: Quick_settlement

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A rapid but high-depth settlement generator that provides a visual reference image up front and organizes key NPCs into a structured, goal-oriented table.

## 2. Core Inputs
- **Settlement Type & Setting:** (e.g., City in a Forest).
- **World Genre:** (e.g., High Fantasy).
- **Utmost Priority Details:** User-defined unique hooks.
- **Section-Specific Creative Codes:** AC&OT, MS&L, D&T, C&L, IN, C&S, ND, PD (Prompt Driven).

## 3. UI Requirements
- **Visual Lead Dashboard:** Generates and displays the DALL-E reference image before the full text is drafted.
- **Priority Directive Input:** A prominent text area for the "UTMOST PRIORITY" elements.
- **Categorized Narrative Editors:** 10+ sections (Overview, Geography, Society, etc.) with integrated "Creative Code" indicators.
- **NPC Goal Matrix:** A dedicated table editor for Prominent Individuals (Name, Role, Description, Goal).
- **Adventure Hook Constructor:** Fields for Mysteries, Intrigue, Hunts, Treasures, and Dilemmas.

## 4. Derived & Automated Fields
- **Image-to-Lore Sync:** Ensures the DALL-E image matches the "Geography and Layout" section.
- **Table-to-Narrative Linker:** Automatically populates individual descriptions based on the NPC table data.
- **GM Strategy Engine:** Generates campaign-specific advice based on the interaction of all prior sections.
- **Style Enforcer:** Automatically applies different formatting (C&S, D&T) to the output based on pre-set script requirements.

## 5. Exports & Integration
- Interactive World-Building Doc (Markdown).
- NPC Reference Table (CSV/VTT).
- DALL-E Reference Image (PNG).
- GM Quick-Reference Cheat Sheet.
