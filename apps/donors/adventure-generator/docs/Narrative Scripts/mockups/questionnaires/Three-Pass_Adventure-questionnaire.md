# UI Questionnaire: Three-Pass Adventure Studio

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A high-fidelity adventure design workspace that implements the "Golden Compass" three-pass methodology, taking a DM from a raw concept to a table-ready, professionally formatted module.

## 2. Core Inputs
- **Pass 1 (Narrative):** Premise, Objective, Rival, Scene Outlines, Branching Paths.
- **Pass 2 (Mechanical):** Challenge DCs, Combat stats, Doomsday Clock, Resource Costs.
- **Pass 3 (Presentation):** Sensory "Read Aloud" text, FM Tip boxes, Visual Layout.

## 3. UI Requirements
- **The Progression Stepper:** A top-level navigation bar representing Pass 1, Pass 2, and Pass 3.
- **Scene Narrative Framework:** A modular editor where scenes are first outlined narratively (Pass 1) before mechanical fields are added (Pass 2).
- **NPC/Rival Dossier:** A dedicated area for evolving antagonists from "Narrative Roles" to "Stat Blocks."
- **Doomsday Clock Visualizer:** A tool to set and track the time-pressure mechanics.
- **The "Read Aloud" Studio:** A sensory-focused text editor for the final atmospheric pass.
- **FM Troubleshooting Console:** A section for "What If?" scenarios and pacing advice.

## 4. Derived & Automated Fields
- **Mechanical Hydrator:** Automatically suggests DC levels (Basic, Critical, Extreme) based on the scene's narrative importance.
- **Bullet-Point Summarizer:** A tool that derives the script-required "Summaries" from the primary narrative text.
- **Usability Auditor:** Flags scenes missing "Transitions," "Objectives," or "NPC Guidance."

## 5. Exports & Integration
- Fully Formatted Adventure PDF/Markdown.
- Interactive Session Dashboard (incorporating Pass 3 guidance).
- VTT Asset Pack (Stats, Maps, Descriptions).
