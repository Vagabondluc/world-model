# UI Questionnaire: campaign_prep

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A master-level campaign architect for conceptualizing long-term RPG experiences. It provides a detailed framework for Setting, Metaplot, Recurring NPCs, and a 3-Act Structure.

## 2. Core Inputs
- **Fantasy Flavor:** (Select/Blend: Dark, Grimdark, High, Low, Portal, Sci-Fi, Sword & Sorcery, Weird).
- **Scale:** Sessions (10-15, 20-30, 50+), Level Range (1-5, 1-10, 1-20), Real-time duration.
- **Tiers of Play:** (Tier 1 Local to Tier 4 Reality-Shaping).
- **Setting Details:** Geography, Politics, Society, Tech/Magic, Culture.
- **Themes:** (3-5 core themes, e.g., Redemption, Corruption, Nature vs Tech).
- **NPC Profiles:** (5-7 NPCs with goals, traits, and connections to metaplot).
- **3-Act Arc:** Specific events, twists, and climaxes for each act.

## 3. UI Requirements
- **Campaign Identity Console:** High-level settings for Flavor, Tier, and Duration.
- **World-Building Studio:** Tabbed editors for Geography, Politics, and Culture.
- **Metaplot Timeline:** A visual board to map out 5-7 major milestones.
- **Cast of Characters Manager:** A dedicated database for 5-7 recurring NPCs.
- **Story Arc Architecture:** A 3-column view (Act 1, 2, 3) to manage escalating stakes.
- **Branching Decision Tool:** A workspace to define 5-7 major player-driven branching points.

## 4. Derived & Automated Fields
- **Consistency Auditor:** Ensures that NPC goals align with the campaign's core themes.
- **Stakes Escalator:** Suggests how to raise the difficulty/consequences as the campaign moves between acts.
- **Backstory Integration Hub:** Suggests specific ways to link PC backgrounds to the established Metaplot milestones.

## 5. Exports & Integration
- Full Campaign Guide (Markdown, ~3000-4000 words).
- NPC Master List.
- 3-Act Timeline Infographic.
- Session 0 Handout.
