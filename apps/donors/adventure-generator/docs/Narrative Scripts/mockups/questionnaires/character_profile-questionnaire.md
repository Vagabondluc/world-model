# UI Questionnaire: Character Portrait & Backstory Suite

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A comprehensive creation environment for players and GMs to develop deep, multi-layered characters, moving from meta-personality archetypes to granular 1500-word backstories and narrative growth arcs.

## 2. Core Inputs
- **Player/Character Archetype:** (Determined via the Personality Questionnaire).
- **Core Narrative Structure:** (Hero's Journey, Story Structure 101, etc.).
- **Character Role:** (Protagonist, Antagonist, Mentor, Supporting).
- **Surface Profile Language:** (Select: Français or English).

## 3. UI Requirements
- **Sensory Personality Wizard:** An interactive immersive questionnaire (5 senses, moral dilemmas) to categorize the character's 'Want' vs 'Avoid' roles.
- **Backstory Studio (1500w):** A sectionalized editor covering Basic Details, Physical Attributes, Clothing/Style, Psychology, and Heritage.
- **Growth Arc Navigator:** A timeline-based tool that maps the character's journey (Ordinary World -> Inciting Incident -> Transformation -> Resolution).
- **French/English Surface Profiler:** A dual-tab output for player-facing summaries (Nom, Occupation, Apparence, Personalité).
- **Social Relationship Web:** A canvas for mapping 'Bonds' and 'Unnatural Study' hooks.

## 4. Derived & Automated Fields
- **Archetype Point Matrix:** Automatically calculates role alignment (e.g., Storyteller vs Meta-gamer) based on questionnaire responses.
- **Catchphrase & Tick Generator:** Suggests unique speech patterns and mannerisms based on the Psychological Profile.
- **Hero's Journey Hook Suggestor:** Generates 1-3 'Inciting Incidents' tailored to the character's 'Biggest Fear' and 'Pride.'

## 5. Exports & Integration
- 1500-word Character Dossier (Markdown).
- Player-Facing Surface Profile (PDF/Markdown Card).
- Character Arc Timeline (SVG/Mermaid).
- VTT Personality Token (JSON).
