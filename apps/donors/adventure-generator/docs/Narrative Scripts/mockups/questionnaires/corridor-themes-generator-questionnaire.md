# UI Questionnaire: corridor-themes-generator

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A rapid generator for "interstitial" dungeon space descriptions (corridors, hallways, passages).

## 2. Core Inputs
- **Dungeon Theme:** (e.g., "Ancient Tomb", "Cyber-Vents").
- **Length/Type:** (e.g., "Short Path", "Intersection", "Grand Hall").
- **Complexity Filter:** Toggle for adding Hazards, Sensory details, or Lore.

## 3. UI Requirements
- **One-Click Generate:** Instant results based on the seed.
- **Result Card:** A formatted text block that is easy to copy-paste.
- **Layered Generation:** Ability to "Reroll sensory details" without changing the base shape.

## 4. Derived & Automated Fields
- **Atmospheric Keywords:** Auto-suggested based on the dungeon's overall history.
