# UI Questionnaire: create_hook

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A versatile scenario hook generator that creates compelling entry points for RPG adventures based on nine distinct hook archetypes.

## 2. Core Inputs
- **Hook Type:** (Select: Patronage, Offer, Confrontation, Dilemma, Twisted, Meaningful, Long-Term, Bait, Treasure Map).
- **Core Scenario Elements:** 
    - Where (Location)?
    - Who (Factions/NPCs)?
    - What (Conflict/Challenge)?
- **Additional Context:** Campaign history or character backgrounds.
- **Engagement Strategy:** How players are expected to interact.

## 3. UI Requirements
- **Hook Archetype Selector:** A visual carousel or list of the 9 hook types with brief descriptions.
- **Scenario Foundation Editor:** Inputs for Location, NPCs, and Conflict.
- **Dynamic Content Builder:** Arches the generation based on selected type (e.g., if "Patronage," it asks for the Patron NPC; if "Twisted," it asks for the Lie).
- **Sensory Depth Control:** Fields for evocative language and multi-sensory descriptions.
- **Actionability Checklist:** Ensures the hook has clear "Next Steps" and "Investigation Points."

## 4. Derived & Automated Fields
- **Impact Predictor:** Suggests potential consequences of player action or inaction based on the conflict.
- **Depth & Intrigue Enhancer:** Proposes hidden agendas or plot twists.
- **Vivid Language Injector:** Automatically upgrades generic descriptions into evocative prose.

## 5. Exports & Integration
- Scenario Hook Page (Markdown).
- DM Quick-Start Card.
- Player-Facing "Rumor/Notice" handout.
