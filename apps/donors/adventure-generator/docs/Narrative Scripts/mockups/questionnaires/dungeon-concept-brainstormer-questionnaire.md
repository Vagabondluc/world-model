# UI Questionnaire: Dungeon Detail & Expansion Suite

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
An advanced toolkit for Dungeon Masters to enhance their dungeons with non-linear mapping (Xandering), sensory corridor themes, and a systematic 10-room design workflow, ensuring both structural complexity and narrative depth.

## 2. Core Inputs
- **Initial Spark:** (Select: Location, Denizen, or Goal as the starting focus).
- **Design Method:** (Select: 10-Room Generator vs. General Brainstorming).
- **Xandering Intensity:** (How many loops, level-skips, and secret paths to include).
- **Global Constraints:** (Ceiling height, light sources, primary materials).

## 3. UI Requirements
- **Concept Brainstorming Pod:** An interactive tool to pair creatures, twist locations, and define 'Magic Item Motivations.'
- **The 10-Room Stepper:** A sequential, 10-step wizard that asks categorized questions (Guardian, Logical Location, Resource Drain, etc.) one at a time.
- **Corridor Theme Matrix:** A sensory editor for sights, sounds, and smells specific to hall-ways.
- **Global Feature Summarizer:** A high-level dashboard to set 'Dungeon-Wide' stats (Climate, Lighting, Architecture).
- **Xander-Check Display:** A structural audit tool that verifies 'Multiple Entrances' and 'Discontinuous Connections.'

## 4. Derived & Automated Fields
- **Logical Follow-Through Engine:** (JP Cooper method) Automatically suggests Room 2 based on the theme in Room 1.
- **Random Corridor Generator:** One-click generation of a 5-10 item table of descriptors and hazards.
- **Structural Complexity Score:** Rates the dungeon's non-linearity based on the 'Xandered' techniques implemented.

## 5. Exports & Integration
- Expanded Dungeon Key (Markdown).
- Corridor Random Table (CSV/Markdown).
- Structural Map (Graph-viz representation of non-linear paths).
