# UI Questionnaire: dungeon-features-summarizer

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A "Common Dungeon Features" cheat sheet generator to maintain consistency across all rooms of a dungeon.

## 2. Core Inputs
- **Base Style:** (e.g., Natural Cave, Worked Stone, Sci-Fi Hull).
- **Illumination:** (e.g., Magical, Torches, Bioluminescent, Pitch Black).
- **Ceiling Height:** (e.g., 10ft, 25ft, Varied).
- **Architecture:** Door types, floor materials, corridor widths.
- **Atmosphere:** Smell, Sound, Temperature.

## 3. UI Requirements
- **Dashboard Interface:** Grouped categories (Illumination, Architecture, Atmosphere).
- **Visual Presets:** Clicking a "Prison" preset auto-fills bars, iron doors, and damp smell.
- **Copy-Paste Section:** A condensed "Reference Card" view for easy inclusion in DM notes.

## 4. Derived & Automated Fields
- **Consistency Check:** If "Pitch Black" is selected, "Reading" based puzzles might trigger a warning.

## 5. Exports & Integration
- Export to Foundry VTT "Journal Entry".
- Markdown Cheat Sheet.
