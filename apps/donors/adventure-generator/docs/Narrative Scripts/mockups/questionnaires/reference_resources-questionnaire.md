# UI Questionnaire: Narrative Engine & Tactical Lexicon

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## 1. Primary Purpose
A comprehensive reference and injection tool for GMs to browse plot structures, apply tactical roles to enemies, and modify scenes with narrative twists in real-time.

## 2. Core Inputs
- **Narrative Need:** (Plot Hook, Mid-scene Twist, Character Arc Shift).
- **Tactical Context:** (Ranged vs Melee balance, Group vs Solo challenge).
- **Atmospheric Goal:** (Tone: Grim, Whimsical, Suspenseful).
- **Scale:** (Personal, Communal, Regional, Global).

## 3. UI Requirements
- **The Plot Engine Browser:** A searchable library of the 34 "Core Plot Engines" (e.g., Delver's Delight, Manhunt) with one-click injection into current session notes.
- **Twist Deck Module:** An interactive card-like selector for the 18 "Universal Twists" (e.g., Sympathetic Villain, Another Group Failed First).
- **Creature Role Injector:** A toggle interface to overlay "Creature Roles" (Brute, Artillery, etc.) onto any loaded NPC or Monster statblock.
- **Workflow Operator Pane:** Tools for "Hook vs Meat" transitions and "Reversal" of moral roles.
- **Event & Tone Matrix:** A generator for Wilderness/Urban/Travel events filtered by "Tone & Mood" (Step 199 of reference).

## 4. Derived & Automated Fields
- **Tension Calculator:** Estimates the "Narrative Pressure" based on the selected plot engine and twist combinations.
- **Tactical Synergy Checker:** Recommends creature pairings (e.g., "Ambusher works best with 2 Brutes").
- **Constraint Generator:** Automatically suggests "Moral or Functional Constraints" to force creative player solutions.

## 5. Exports & Integration
- Narrative Scene Manifest (Markdown).
- Tactical Combat Profile (JSON).
- GM Reference Quick-Cards (SVG).
- World-Building Twist Report.
