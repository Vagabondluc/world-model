## 1. CreateRiddle (The Riddle Forge)

**1. What is the primary input for riddle generation?**
The Answer (provided by the DM).

**2. What are the two mandatory difficulty levels?**
- Easy Version
- Harder Version

**3. What critical explanation must the AI provide?**
A detailed breakdown of *why* the answer is correct for both versions.

**4. What narrative "Flavor" can be applied?**
(Optional enhancement) Themes like Grave-marker, Sphinx, Mad Wizard, or Cryptic Door.

## 2. CreateToken (The Token Studio)

**1. What is the required perspective for the image?**
Top-down aerial view (directly from above).

**2. What is the intended use-case for these assets?**
2D game art with transparent backgrounds, suitable for battlemaps or D&D virtual table-tops.

**3. What are the key descriptive elements?**
Vivid colors and attire details of the specific character.

**4. How does the UI handle the "Prompt Template"?**
By replacing the `[visual description]` bracket with user input while maintaining the "vivid colors, 2D game art, transparent background" boilerplate.
