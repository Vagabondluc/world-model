# UI Explanation: AI Storyteller Dashboard

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## Component Architecture
- **Storyteller Narrative Feed:** A rich-text display for the AI's "vivid depictions." It includes "Subjective Layers" which can be toggled to provide deeper immersion into the character's internal state.
- **Environment Logic Drawer:** A persistent sidebar that tracks "Established Facts" (e.g., Setting consistency). This ensures that neither the AI nor the user introduces elements that contradict the established universe (like a police force on a deserted freighter).
- **Task Evaluation Engine:** A UI module where the Storyteller classifies user actions as "Effortless" or "Complex." It displays the evaluated difficulty and the results of ability-based successes or failures.
- **Conflict Sequencing Module:** A mode-switch that reorganizes the UI for tactical encounters, focusing on the "sequence and outcome of events" as described in the script.
- **Protagonist Intention Studio:** The user's input area, optimized for describing both "intentions" (the goal) and "actions" (the method).

## Interaction Logic
- **Constraint-Based Outcome Resolution:** The UI guides the Storyteller through evaluating the user's intent vs. character abilities, turning high-level goals into concrete narrative consequences.
- **Dynamic Tension Tracking:** A visual "Tension Meter" that the AI adjusts to signal narrative stakes and suspense levels.
- **Vividness Overlays:** A button to generate "Layered & Explicit" descriptions of a scene, following the script's requirement for clear, non-ambiguous environment mapping.

## Visual Design
- **Cinematic & Immersive:** The UI uses a "Terminal/Console" aesthetic for Sci-Fi settings or a "Scribe's Ledger" for Fantasy, adapting to the "Active Context."
- **Dual-Focus Layout:** Balances the "Story Feed" (Roleplay) with the "Environment & Logic" (Rules/Consistency) panels to keep the GMing side of the AI transparent.
- **Text-Centric Clarity:** Priorities readability and high-contrast text to support the "Storyteller" role, with subtle animations for narrative "Tension" shifts.
