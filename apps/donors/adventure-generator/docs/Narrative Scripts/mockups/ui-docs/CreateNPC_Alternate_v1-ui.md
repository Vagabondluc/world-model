# UI Explanation: Alternate NPC Template Builder

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## Component Architecture
- **Identity & Orientation Studio:** A localized editor that handles Race, Name, Gender, and Sexual Preference. It prioritizes inclusivity and clarity.
- **Physicality Panel:** A specialized set of fields for the script's requirement for "Skin Color, Height, Weight" and general descriptors.
- **The "Career Origin" Editor:** A dual-purpose module that links the NPC's "Job" with their "Reason for taking it," creating a more cohesive backstory.
- **Fantasy Hook & Secret Console:** Two high-impact text areas focused on D&D-style utility—secrets to keep the players engaged and hooks to start adventures.
- **Representative Analogy Tool:** A unique UI element where the DM can record a "Famous Character" comparison, drastically reducing the mental load of roleplaying a new character.

## Interaction Logic
- **Visual Description Synthesis:** As details are entered into the Height/Weight/Skin fields, the UI suggests a descriptive sentence for the DM to read aloud.
- **Secret-to-Hook Linking:** The UI encourages the "Secret" and "Quest Hook" to be narrative twins—the hook often leads the players to discovering the secret.
- **Job-Specific Inventory:** Selecting a job (e.g., "Cartographer") pre-populates suggested items they would "carry on themselves."

## Visual Design
- **Structured & Encyclopedic:** The UI uses a "Tabular Dossier" aesthetic, but with a softer, more modern interface than the "Detailed NPC Architect."
- **Focus on Continuity:** Fields like "Comparison" are given visual prominence (e.g., a card-like frame) because they are the most useful shortcuts for the DM.
- **Roleplay-First Highlight:** The "Speech Example" is placed in a high-contrast box to ensure it's the first thing the DM sees when preparing to speak as the NPC.
