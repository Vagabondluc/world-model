# UI Explanation: Encounter_genric_beta

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## Component Architecture
- **Realm Themed Interface:** A visual selector for "Mystical Realms" that changes the overall UI aesthetic (e.g., starry astral background vs shadow-wrapped border).
- **Narrative Word-Count Console:** A central text area with a prominent word counter that changes color (Yellow -> Green -> Red) as the user approaches the "500-word" sweet spot.
- **Twin outcome Panels:** Dedicated fields for "Success Consequences" and "Failure Consequences", presented side-by-side to ensure balance.
- **NPC Profile Card:** A small, detailed block covering "Motive", "Goal", and "Cunning Tactics".

## Interaction Logic
- **Success-Based XP Calculations:** The XP fields are mathematically linked. If "Success [800]" is entered, "Failure" can be set as a percentage (e.g., 25%) or a manual value.
- **Sectioned Formatting:** The UI automatically injects section headers (e.g., "The Environment", "The Challenge") into the final export to ensure the "Methodical Arrangement" required by the script.
- **AI Inspiration Chips:** Suggests "Mystical Realms" or "Environmental Twists" based on a single seed word.

## Visual Design
- **High-Phantasy Aesthetic:** Uses glowing borders, serif typography, and translucent panels to evoke a "captivating story" feel.
- **Status Indicators:** Clear badges for "Environmental Mechanics" and "Combat Stats Reference" ensure the DM doesn't miss key rules during play.
- **Export Preview:** A stylized view showing how the content will look in a digital adventure log.
