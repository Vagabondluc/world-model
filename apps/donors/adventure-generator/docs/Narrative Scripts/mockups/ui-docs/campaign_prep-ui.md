# UI Explanation: campaign_prep

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## Component Architecture
- **Metaplot Control Center:** A macro-level dashboard to set global campaign parameters like Fantasy Flavor, Tiers of Play (Lv 1-20), and intended session length.
- **Modular Design Studio:** A multi-tabbed interface that breaks the massive campaign design process into manageable chunks (World, Metaplot, Cast, Arc, PC Integration).
- **Visual Arc Architect:** A high-level 3-Act planner that helps the DM manage the escalation of stakes and the "Character Development Opportunities" required by the script.
- **NPC Relationship Matrix:** A detailed database for the 5-7 recurring characters, ensuring their goals, quirks, and connections to the metaplot are always visible.
- **Branching Point Editor:** A dedicated workspace for identifying the 5-7 "Flexibility" points where player choices can fundamentally change the campaign's direction.

## Interaction Logic
- **Theme-to-Scene Weaving:** When the DM selects a theme (e.g., Sacrifice), the UI suggests specific events in the 3-Act Arc that can test this theme for the players.
- **PC Integration Helper:** Provides a list of 5-7 "Hook Toggles" (e.g., Prophecy, Faction, Family) to easily link player backgrounds to the core campaign Metaplot.
- **Adaptation Logic Builder:** A "What If?" panel that allows the DM to plan for potential character deaths or major failures at each branching point.

## Visual Design
- **Authoritative & Strategic:** A dense, high-information layout that feels like a campaign management dashboard or a professional storyboard.
- **Chronological Flow:** The UI uses horizontal progression for Act-based planning and vertical progression for deep-dives into specific NPC or Location details.
- **Creative Sidebars:** Includes callout boxes for "Variant Ideas" and "Inspirational Content," exactly as requested in the additional instructions of the script.
