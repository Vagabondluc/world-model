# UI Explanation: fantastic_location

**Version:** 0.1.0
**Status:** Draft
**Owner:** TBD
**Last Updated:** 2026-02-04


## Component Architecture
- **Evocative Naming Tool:** A high-impact title generator that ensures names are brief (3-5 words) but descriptive.
- **Triptych Aspect Editor:** Specifically limits the user to "Exactly Three" aspects. This design choice prevents "Feature Creep" and forces focus on the three most interactive elements of the scene.
- **Sensory Palette Modal:** An interactive checklist for the 5 senses. It requires descriptions for at least three to ensure the scene feels "Alive."
- **Scale & Chronology Dashboard:** A simple, illustrative workspace for defining the physical dimensions and the primordial age of the location.
- **Background Context Mapper:** A logic-builder that links the location's lore to common RPG character archetypes (Archeology, Religious training, Arcane knowledge).
- **Concise Narrative Preview:** A real-time word-counter and text-block previewer to help the DM stay within the 300-400 word limit.

## Interaction Logic
- **Aspect-Driven Prompts:** Each aspect entered by the user generates potential "Interaction" suggestions (e.g., entering "Cracked Geode" might suggest "Mining for crystals" or "Magical resonance check").
- **Sensory Re-skinning:** Changing the theme of the location (from "Bone" to "Crystal") automatically updates the suggested sensory vocabulary.
- **Export to Scene Card:** Converts the dossier into a compact, printable format designed for a DM's binder or VTT macro.

## Visual Design
- **Cinematic & Moody:** Uses high-contrast typography and plenty of whitespace to emphasize the "Fantastic" nature of the content.
- **Focus on Imagery:** Icons for the 5 senses (Ear, Eye, Nose, Hand, Tongue) provide clear visual cues for narrative preparation.
- **Pacing HUD:** The word counter at the bottom acts as a "Pacing Guide," encouraging brevity and punchy descriptions.
