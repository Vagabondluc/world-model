# Mappa Imperium

**A collaborative digital worldbuilding application.**

Mappa Imperium is a web-based tool designed to bring the experience of the tabletop worldbuilding game to a digital format. It guides players through a structured, six-era process of creating a unique fantasy world, complete with its own geography, mythology, empires, and history.

---

## 🚀 Current Status: Major Refactor - Phase 3 Complete

This project is currently undergoing a significant architectural refactor to a modern, professional tech stack. Phase 3, which rebuilds the modular debug system on top of the new Zustand store, is now complete.

-   **Frontend**: React with TypeScript, built with Vite.
-   **State Management**: Zustand.
-   **Styling**: Tailwind CSS.

## 🛠️ Getting Started

### Prerequisites

-   [Node.js](https://nodejs.org/) (version 18.x or higher)
-   `npm` (comes with Node.js)

### Installation

1.  Clone the repository:
    ```bash
    git clone [repository-url]
    ```
2.  Navigate to the project directory:
    ```bash
    cd mappa-imperium
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Development Server

To start the local development server, run:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.
