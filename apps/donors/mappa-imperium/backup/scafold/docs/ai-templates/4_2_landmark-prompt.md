# Fantastic Landmark Generator

## Required Input
- Nearby Tribe's Race
- General Geographic Region (forest, coast, mountains, etc.)
- Any significant faction traits (optional)

## For Each Landmark, I Will Generate:

### Physical Description
A detailed narrative of the landmark's:
- Scale and scope
- Distinctive features
- Environmental impact
- Unusual properties
- Seasonal changes

### Tribal Integration
How the nearby tribe:
- Interprets the landmark
- Uses its features
- Has adapted to its presence
- Incorporates it into their culture

### Current Status
- Active phenomena
- Recent changes
- Immediate dangers/opportunities
- Resource potential

### Story Hooks
- Local legends
- Unexplained aspects
- Current conflicts
- Future possibilities

1. Each Landmark Must Have:
   - Unique physical characteristics
   - Specific environmental effects
   - Named tribal figures associated with it
   - Clear cultural impact on the tribe
   - Concrete current situation
   - Multiple potential plot hooks

2. Details Should Include:
   - Exact measurements where relevant
   - Specific materials and compositions
   - Seasonal or cyclical changes
   - Sensory information (sounds, smells, etc.)
   - Resource implications
   - Safety considerations

3. Tribal Connections Should Show:
   - How they discovered it
   - How they use it
   - How it changed them
   - Current debates/conflicts
   - Future plans

4. Landmarks Should:
   - Feel distinct from each other
   - Connect logically to the environment
