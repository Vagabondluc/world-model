# Current Implementation Documentation

The documents in this folder describe the Mappa Imperium application as it is **currently implemented**.

This version of the application is a client-side React application with state managed locally. Session persistence is handled via a JSON import/export system, and multiplayer interaction is simulated locally.

Refer to these documents for an accurate understanding of the existing codebase. For the future vision of the project, see the `/docs/roadmap` directory.
