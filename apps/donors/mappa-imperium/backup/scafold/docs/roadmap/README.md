# Project Roadmap & Target Architecture

The documents in this folder describe the **future vision and target architecture** for Mappa Imperium.

This includes plans for a full-stack application with a dedicated backend, a database for persistent storage, and true real-time, multi-player collaboration via WebSockets.

These documents serve as a guide for future development. For an accurate description of the current, simpler implementation, please see the `/docs/current` directory.
