# Event Form Specifications

This directory contains the detailed form specifications for each event in Eras IV, V, and VI. These documents serve as the blueprint for building the UI and AI integration for the game's dynamic event system.

Each file corresponds to a specific dice roll outcome from the official Mappa Imperium rulebook. The structure of these specifications is based on the comprehensive example provided in `/example/complete_event_forms_spec.md`.

## Purpose

- **UI Development**: Provides a clear contract for frontend developers to build the dynamic forms required for each unique event.
- **AI Integration**: Outlines the specific data points that need to be collected and sent to the AI service for generating contextual narratives.
- **Game Logic**: Ensures that the application's event handling remains consistent with the tabletop game's rules and mechanics.
- **Quality Assurance**: Serves as a checklist for testing to ensure all event paths are implemented correctly.
