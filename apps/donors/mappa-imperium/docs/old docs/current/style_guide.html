Design Principles

Color Usage

  • Amber: Primary actions, current states, focus
  • Green: Completed states, success messages
  • Blue: Available actions, information
  • Gray: Disabled/locked states

Spacing

  • 4px: Small gaps, badges
  • 8px: Form elements, button padding
  • 16px: Card padding, section gaps
  • 32px: Major section spacing
