# Framework and Axes

## Purpose
Define the conceptual model of the extended Whittaker system captured in `Google Gemini.pdf`.

## Core Model
Traditional Whittaker coordinates are extended from 2D climate mapping to multi-axial world generation.

Base axes:
- Temperature
- Precipitation

Extended axes:
- `Z` (Strata / Verticality)
- `W` (Spirit / Metaphysical law set)

## Axis Semantics

## Z-Axis: Strata
Represents vertical adventure domains:
- `Aero` (sky)
- `Terra` (surface)
- `Litho` (upper dark/crust interior)
- `Abyssal` (deep dark/core-root depths)

Design effect:
- Same climate band can produce radically different play-space when mapped to another stratum.

## W-Axis: Spirit
Represents plane-law overlays:
- Material (neutral/ecological realism)
- Feywild (vitality/excess/predatory whimsy)
- Shadowfell (entropy/decay/psychological pressure)

Design effect:
- Biome identity is not purely physical; it is also metaphysical.

## Composition Rule
A playable biome cell is generated by combining:
1. Climate zone baseline
2. Stratum transform (`Z`)
3. Spirit transform (`W`)
4. Local trope/hazard/resource/inhabitant constraints

This yields a matrix of distinct biome variants rather than one biome per climate.
