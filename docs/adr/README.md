# ADR Index

Use this folder for architecture decision records that need a permanent rationale.

Suggested ADR topics:

- copy-only donor adapter policy
- canonical model ownership
- promoted schema provenance
- final app shell and mode structure
- migration and cutover rules
- release boundary rules

Each ADR should explain:

1. the decision
2. the options considered
3. the tradeoffs
4. the chosen outcome
5. the consequence for the codebase
