# Unified App User Guide

## Open a bundle

- Use the `Open bundle` control in the shell context bar.
- Load a canonical bundle JSON file produced by the app or by migration tooling.

## Navigate the app

- `World` is the primary public view for world editing and browsing.
- `Story` groups narrative and progression work over the same canonical bundle.
- `Schema` exposes contract, adapter, import, and report surfaces.
- Legacy route aliases remain available for compatibility, but they are not the primary navigation.

## Use modal tools

- Open the shell tools menu for shared wizards and generators.
- Use contextual buttons inside each tab when a modal tool belongs to the current workflow.
- Modal tools include world creation, entity creation, Markov naming, city generation, import preview, and migration report viewing.

## Save and reload

- Use `Save bundle` to export the current canonical bundle to JSON.
- Reopen the exported file later to continue from the same canonical state.
- Overlay state such as selected panes and drawers is not part of the saved bundle.
